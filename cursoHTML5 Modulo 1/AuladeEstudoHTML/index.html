Negrito e Italico

Negrito/Destaque

usamos a tag <b>para deixar as palavras em negrito</b>
tag B (é um tag sem Semantica).
usamos <strong> que é uma tag semantica para deixar em negrito </strong>

Italico/Enfase

usamos a tag <i> para deixar as palavras em italico (nao semantica) </i>
Apertar ctrl + ship + p com a frase selecionada e ir em Envelope com a abreviatura (Wrap with Abbreviation) para ganhar tempo quando for colocar a tag i.
usamos <em> para deixar as palavras em italico (com semantica) </em>

Texto Marcado

usamos a tag <mark> para marcar as palavras como um marca texto </mark>
para mudar a cor utiliza CSS Criado antes do body para mudar em mais de uma
marcação para a mesma cor.
Definindo a cor usando CSS

Texto grande/pequeno

usamos <big> para deixar o texto grande <big>
big é uma tag obsoleta porem ainda funciona
usamos <small> para deixar o texto pequeno <small>

Texto deletado

usamos <del> para indicar um texto excluido que nao deve ser considerado </del>

Texto inserido

usamos <ins> para indicar um texto inserido e inidicar que ele foi adicionado depois </ins>
Existem tambem o sublinhado com a tag U (nao semantica) usamos <u> para sublinhar o texto como a tag ins (semantica) <u>

Texto sobrescrito

para inserir dados do tipo x^20+3
usamos a tag no numero que queremos elevar x <sup>20</sup>+3 = x20+3

Texto subscrito

para inserir coisa do tipo H2O usamos a tag no numero que queremos deixar em baixo
H<sub>2</sub>O = H2O

Trbalhando com listas

Listas ordenadas

usamos <ol> e <li> na frente de cada linha nao sendo necessario usar o /li
para usar outro tipo de referencia na lista, setar o type em ol type = "".
<ol>

  1. para colocar uma lista ordenada
  2. como no exemplo
  3. ol para lista ordenada e li para listar
</ol>

Listas não ordenadas

usamos <ul> e <li> na frente de cada linha nao sendo necessario usar o /li
para usar outro tipo de referencia na lista, setar o type em ul type = "".
<ul>

  • para colocar uma lista não ordenada
  • como no exemplo
  • ul para lista não ordenada e li para listar
</ul>
